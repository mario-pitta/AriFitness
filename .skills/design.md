# Design System Document

This document outlines the core design principles and token values for our digital products.

## Theme Configuration

Our design system is configured with the following key characteristics:

* **Color Mode**: `dark`
* **Headline Font**: `spaceGrotesk`
* **Body Font**: `inter`
* **Label Font**: `inter`
* **Roundedness**: Elements feature a `moderate` (2) level of roundedness.
* **Spacing**: The layout utilizes `normal` (2) spacing, providing a balanced visual density.

## Color Palette

The chosen color palette supports our brand identity and ensures a consistent visual experience across all interfaces.

* **Primary Color**: `#2D8A5E` - Our brand's most distinctive color, used for key interactive elements.
* **Secondary Color**: `#475569` - A supporting color for less prominent UI components.
* **Tertiary Color**: `#10B981` - An accent color for highlights and decorative elements.
* **Neutral Color**: `#111827` - The base color for backgrounds and surfaces, providing a solid foundation for the UI.

## Typography

### Headlines

Headlines are set in `spaceGrotesk`, chosen for its distinct character and readability, ensuring strong visual hierarchy.

### Body Text

Body text uses `inter`, a highly legible and versatile typeface that provides clarity for longer content.

### Labels

Labels also use `inter` to maintain consistency with body text, ensuring clear and readable annotations and form labels.

## Shape and Form

User interface elements are designed with a `moderate` (2) level of roundedness, striking a balance between modern aesthetics and approachability. This ensures a consistent visual feel while avoiding overly sharp or completely pill-shaped forms.

## Spacing

A `normal` (2) spacing scale is applied throughout the interface. This provides adequate whitespace for readability and visual separation between elements without making the interface feel too dense or overly spacious. It supports an organized and user-friendly layout.
